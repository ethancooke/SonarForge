# Assets

Place app icons, menu bar icons, and other image assets here.

Recommended:
- `AppIcon.appiconset` (generated via Xcode asset catalog or Icon Composer / `iconutil`).
- A simple monochrome template image for the status item (e.g. `status-icon.pdf` or SF Symbol usage in code).

For distribution builds, the icon must be high quality at all required sizes.
